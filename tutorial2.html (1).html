Lesson 2: Variables

One of the most important things to know when using Python is how to create variables, which are placeholders for data so you can store them for later use.

Creating a variable is easy to do, you make a name that you want to assign the value and use an equal sign, then write the code you would like to be attached to the name. This creates a variable that is stored and can be called upon whenever the name is used in a line of code.

Variables can be used for any data type, so it makes it incredibly diverse for people to use for a variety of different situations. Here is an example of a string being assigned to a variable.

Input | x = "Potato"

print(x)

Output | Potato

An integer is treated as a numerical value as opposed to a written one, like if you were to put two values in a calculator in order to figure out the result of a calculation. Creating an integer is as simple as using the function "int()" which only works if a number is used unless the value written in the brackets is a variable with a numerical value.

Here is an example of an integer being used to give a variable a numerical value:

Input | x = 2

y = 3

print(x + y)

Output | 5
