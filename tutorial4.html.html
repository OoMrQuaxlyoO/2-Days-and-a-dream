Lesson 4: Lists

Variables were useful for keeping functions and values under a name that could be called in the future, but another way of organising information to be used at a later time is by making a list.

------------------------------------------------------------------

Lists

A list is an index of data that is stored under a variable that you can extract a specific piece of data from by specifying which one you want.

Here is a demonstration of a list being created and used to get a specified result from the created list:

Input: | x = [1, 2, 3]

print(x[0])

Output: | 1
