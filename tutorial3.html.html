Lesson 3: For & While Loops

------------------------------------------------------------------

While Loops

While loop are used to initiate code repeatedly until a determined condition is met. When the condition becomes false, the loop breaks and the line under the loop is executed. While loops are handy for when you set up a system where the user must enter data that must meet the conditions set in a while loop in order for them to progress, an example of this could be for creating an account for a website and you needing to create a password containing letters and/or numbers.

This example shows how you can limit a user to the size of a number that they can pick, by limiting them from any number from one to ten.

x = input("Enter a number 1-10")

while x <= "0" or x >= "11":

For Loops

For loops are slightly different to a while loop. Where while loops required you to break a certain condition before the next line of code can run, for loops will run through a list when started until every possible result is accounted for, terminating afterwards bringing the code back to the line below. This is useful whenever you make a list of values or names and want them to be outputted in a program.

Here's an example of a for loop in action:

Input: | x = "Duck"

for i in x:

‎ ‎ ‎ ‎‎ ‎ ‎ ‎print(i)

Output: | D u c k
