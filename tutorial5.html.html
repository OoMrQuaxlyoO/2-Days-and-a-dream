Lesson 5: If and Else statements

For our final lesson, we will teach you the differences between If and Else statements. These statements are used when you want to create multiple different conditions when making an input

------------------------------------------------------------------

If statements

An if statement is used to make a condition that the user's or program's input will need to follow, if this condition is met then the program will continue to run the line below the if statement.

Else statements

An else statement is used to define what happens when the input does not meet any of the if statement conditions, the line below the else statement is what is used if no conditions were met.

When using if statements, it is important that you always use an else statement afterwards. If your code does not include an else statement then the code will not output a result if your input did not meet the criteria for an if statement. Here is an example of how if and else statements are used to authorise an input by setting a condition.

Input: | x = 1

‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ if (x > 0):

‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ print("x is positive")

‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ else:

‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ ‎ print("x is 0 or negative")

Output: | # larger than 0

print("x is positive")

# 0 or lower

print("x is 0 or negative")
