Lesson 1: Strings & Integers

Let's start things off with something simple that you will need to know when coding in Python. Python has a lot of values called "Data Types" which determine how code is read, they are used to make sure that data is processed in a way that is suitable for the algorithm that the programmer intends to create. For this lesson, we will be diverting our focus to the two most commonly used data types in Python algorithms: Strings and Integers.

------------------------------------------------------------------

Strings

A string is typically treated as a piece of written text, kind of like if you were to type in a message into a friend group. To create a string, all you need to do is make sure whatever you have written is in quotation marks. ("") In order to print out what you have written in your string, you will need to use the function "print()" and insert the string inside the brackets.

Here's an example of a string that is being printed using the print() function. The Input is the algorithm whereas the Output is what is created when the algorithm is ran.

Input: | print("Hello World!")

Output: | Hello World!
